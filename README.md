This is a simple, intuitive web-based personal finance application designed to help you track daily income and expenses, manage budgets, and understand spending habits.
This project is built using pure HTML5, CSS3, and JavaScript without any external frameworks or dependencies.

FEATURES: 
Dual Tracking: Log both income and expenses with ease.
Transaction History: View a chronological list of all financial entries.
Easy Deletion: Remove mistakes instantly with a single click.
Local Storage: Keeps your financial data intact even after closing or refreshing the browser.
Responsive Design: Clean UI that works seamlessly on desktop, tablet, and mobile screens.
